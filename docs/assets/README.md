# Assets

This folder contains reference media for The Biz Gift design and development.

## Folder Structure

- `moodboard/` — Brand direction references, colour palettes, texture studies
- `references/` — Logo files, brand guidelines PDFs, competitive screenshots
- `photography/` — Approved product and lifestyle photography (not committed to repo; store in shared drive)

## Usage Guidelines

- Never commit large image files to Git. Use Git LFS or external storage (Cloudinary, S3).
- Reference files in documentation with relative paths: `../assets/references/TBG Logo New.png`.
- Photography should follow the image standards defined in `design.md` and `content-architecture.md`.
